ICON PLACEHOLDER

The extension requires 3 icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

For now, you can:
1. Create simple PNG files with the RMG logo or Ollama logo
2. Use any 16x16, 48x48, and 128x128 PNG images as placeholders
3. Or download icons from: https://www.flaticon.com/

Suggested design:
- RMG samurai sword icon
- Red/black color scheme
- Simple, recognizable design

The extension will work without icons, but Chrome will show a default puzzle piece icon.
